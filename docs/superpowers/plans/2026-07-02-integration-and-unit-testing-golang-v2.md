# integration-and-unit-testing-golang-v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a complete, reusable Go service template that demonstrates hexagonal architecture plus unit tests (Ginkgo/Gomega BDD **and** table-driven testify) and a working testcontainers-based integration suite — structure only, no real business logic.

**Architecture:** Hexagonal / ports-and-adapters. Inward-pointing dependencies. HTTP handlers (Echo, generated by oapi-codegen) and a Kafka consumer are driving ports; a Postgres repository (sqlboiler models), a Kafka producer, and an in-memory cache are driven adapters. Application core is split CQRS-lite into `services/queries` and `services/commands`. A composition root in `internal/server` wires everything.

**Tech Stack:** Go 1.26, Echo v4, oapi-codegen v2, sqlboiler v4 + lib/pq, dbmate (migrations), segmentio/kafka-go, Ginkgo v2 + Gomega, go.uber.org/mock (gomock), DATA-DOG/go-sqlmock, stretchr/testify, testcontainers-go, spf13/viper.

// Package retry provides a minimal retry helper.
package retry

// Do calls fn up to attempts times, returning nil on the first success or the
// last error once attempts are exhausted. attempts <= 0 is treated as 1.
func Do(attempts int, fn func() error) error {
	if attempts < 1 {
		attempts = 1
	}
	var err error
	for i := 0; i < attempts; i++ {
		if err = fn(); err == nil {
			return nil
		}
	}
	return err
}
```

package retry_test

import (
	"errors"
	"testing"

	"github.com/stretchr/testify/assert"
	"github.com/stretchr/testify/require"

	"github.com/sorrawichyooboon/integration-and-unit-testing-golang-v2/pkg/retry"
)

func TestDo_TableDriven(t *testing.T) {
	tests := []struct {
		name       string
		attempts   int
		failFirstN int   // fn fails on the first N calls, then succeeds
		wantCalls  int
		wantErr    bool
	}{
		{name: "succeeds first try", attempts: 3, failFirstN: 0, wantCalls: 1, wantErr: false},
		{name: "succeeds after retries", attempts: 3, failFirstN: 2, wantCalls: 3, wantErr: false},
		{name: "exhausts attempts", attempts: 2, failFirstN: 5, wantCalls: 2, wantErr: true},
		{name: "zero attempts treated as one", attempts: 0, failFirstN: 5, wantCalls: 1, wantErr: true},
	}

	for _, tt := range tests {
		t.Run(tt.name, func(t *testing.T) {
			calls := 0
			err := retry.Do(tt.attempts, func() error {
				calls++
				if calls <= tt.failFirstN {
					return errors.New("transient")
				}
				return nil
			})

			assert.Equal(t, tt.wantCalls, calls)
			if tt.wantErr {
				require.Error(t, err)
			} else {
				require.NoError(t, err)
			}
		})
	}
}
```

// Package errs defines the application error catalog and helpers.
package errs

import "fmt"

// AppError is a coded application error.
type AppError struct {
	Code    string
	Message string
}

func (e AppError) Error() string { return fmt.Sprintf("%s: %s", e.Code, e.Message) }

// Get returns a copy so callers can compare without mutating the catalog var.
func (e AppError) Get() AppError { return e }

// Catalog. ORD-E-4xxx validation/HTTP, ORD-E-5xxx system, ORD-E-6xxx business.
var (
	ErrValidation = AppError{Code: "ORD-E-4000", Message: "invalid request"}
	ErrNotFound   = AppError{Code: "ORD-E-4004", Message: "resource not found"}
	ErrDatabase   = AppError{Code: "ORD-E-5000", Message: "database error"}
	ErrUpstream   = AppError{Code: "ORD-E-5001", Message: "upstream error"}
	ErrUnknown    = AppError{Code: "ORD-E-9999", Message: "unknown error"}
)

// AsAppError reports whether err is an AppError and returns it.
func AsAppError(err error) (AppError, bool) {
	ae, ok := err.(AppError)
	return ae, ok
}
```

package errs

import "net/http"

// ErrorResponse is the JSON body returned for errors.
type ErrorResponse struct {
	Code    string `json:"code"`
	Message string `json:"message"`
}

// NewErrorResponse builds a response DTO from any error.
func NewErrorResponse(err error) ErrorResponse {
	if ae, ok := AsAppError(err); ok {
		return ErrorResponse{Code: ae.Code, Message: ae.Message}
	}
	return ErrorResponse{Code: ErrUnknown.Code, Message: ErrUnknown.Message}
}

var errorStatusMap = map[string]int{
	ErrValidation.Code: http.StatusBadRequest,
	ErrNotFound.Code:   http.StatusNotFound,
	ErrDatabase.Code:   http.StatusInternalServerError,
	ErrUpstream.Code:   http.StatusBadGateway,
	ErrUnknown.Code:    http.StatusInternalServerError,
}

// ErrorToStatusCode maps an error to an HTTP status (500 for unmapped).
func ErrorToStatusCode(err error) int {
	if ae, ok := AsAppError(err); ok {
		if code, found := errorStatusMap[ae.Code]; found {
			return code
		}
	}
	return http.StatusInternalServerError
}
```

type ServerConfig struct {
	Port           int `mapstructure:"port"`
	TimeoutSeconds int `mapstructure:"timeoutSeconds"`
}

type DatabaseConfig struct {
	URL string `mapstructure:"url"`
}

type KafkaConfig struct {
	Brokers           []string `mapstructure:"brokers"`
	OrderCreatedTopic string   `mapstructure:"orderCreatedTopic"`

type Config struct {
	Server   ServerConfig   `mapstructure:"server"`

// Current is the process-wide config, populated by Init.
var Current Config

type options struct{ basePath string }

// Option customizes Init.

	for _, fn := range opts {
		fn(&o)
	}

	name := "config"

		name = "config." + env
	}

	v := viper.New()
	v.SetConfigName(name)
	v.SetConfigType("json")
	v.AddConfigPath(o.basePath)
	if err := v.ReadInConfig(); err != nil {
		panic("config: read " + name + ": " + err.Error())
	}
	var c Config
	if err := v.Unmarshal(&c); err != nil {
		panic("config: unmarshal: " + err.Error())

	"context"
	"database/sql"
	"errors"

	"github.com/sorrawichyooboon/integration-and-unit-testing-golang-v2/internal/domains"
	"github.com/sorrawichyooboon/integration-and-unit-testing-golang-v2/internal/domains/errortype"

//go:generate mockgen -source=./repository.go -destination=./mock/repository.go -package=order_mock
type RepositoryInterface interface {
	Insert(ctx context.Context, o *Order) error
	FindByID(ctx context.Context, id string) (*Order, error)
}

type RepositoryArgs struct {
	Db *domains.DBResolver
}

type Repository struct {
	db *domains.DBResolver
}

func NewRepository(args RepositoryArgs) *Repository {
	return &Repository{db: args.Db}
}

func (r *Repository) Insert(ctx context.Context, o *Order) error {
	const q = `INSERT INTO "orders" ("id","customer_id","amount","status","created_at") VALUES ($1,$2,$3,$4,$5)`
	if _, err := r.db.ExecContext(ctx, q, o.ID, o.CustomerID, o.Amount, o.Status, o.CreatedAt); err != nil {
		return errortype.ErrUnexpected{Err: err}
	}
	return nil
}

func (r *Repository) FindByID(ctx context.Context, id string) (*Order, error) {
	const q = `SELECT "id","customer_id","amount","status","created_at" FROM "orders" WHERE "id" = $1`
	row := r.db.QueryRowContext(ctx, q, id)
	var o Order
	if err := row.Scan(&o.ID, &o.CustomerID, &o.Amount, &o.Status, &o.CreatedAt); err != nil {
		if errors.Is(err, sql.ErrNoRows) {
			return nil, errortype.ErrNotFound
		}
		return nil, errortype.ErrUnexpected{Err: err}
	}
	return &o, nil
}
```

package kafkaclient

import (
	"context"

	"github.com/segmentio/kafka-go"
)

type Consumer struct {
	reader *kafka.Reader
}

func NewConsumer(brokers []string, groupID, topic string) *Consumer {
	return &Consumer{
		reader: kafka.NewReader(kafka.ReaderConfig{
			Brokers: brokers,
			GroupID: groupID,
			Topic:   topic,
		}),
	}
}

// Consume reads messages until ctx is cancelled, invoking handler per message.
// A handler error is returned (the message is not committed).
func (c *Consumer) Consume(ctx context.Context, handler func(ctx context.Context, key, value []byte) error) error {
	for {
		m, err := c.reader.FetchMessage(ctx)
		if err != nil {
			return err // ctx cancelled or reader closed
		}
		if err := handler(ctx, m.Key, m.Value); err != nil {
			return err
		}
		if err := c.reader.CommitMessages(ctx, m); err != nil {
			return err
		}
	}
}

func (c *Consumer) Close() error { return c.reader.Close() }
```

package cacheclient

import "sync"

//go:generate mockgen -source=./cache.go -destination=./mock/cache.go -package=cacheclient_mock
type CacheInterface interface {
	Get(key string) ([]byte, bool)
	Set(key string, value []byte)
}

type MemoryCache struct {
	mu sync.RWMutex
	m  map[string][]byte
}

func NewMemoryCache() *MemoryCache {
	return &MemoryCache{m: make(map[string][]byte)}
}

func (c *MemoryCache) Get(key string) ([]byte, bool) {
	c.mu.RLock()
	defer c.mu.RUnlock()
	v, ok := c.m[key]
	return v, ok
}

func (c *MemoryCache) Set(key string, value []byte) {
	c.mu.Lock()
	defer c.mu.Unlock()
	c.m[key] = value
}

	if cmd.CustomerID == "" || cmd.Amount <= 0 {
		return "", errs.ErrValidation
	}

	o := &order.Order{
		ID:         uuid.NewString(),
		CustomerID: cmd.CustomerID,

		Status:     "NEW",
		CreatedAt:  time.Now().UTC(),

		return "", errs.ErrDatabase

	if err := h.producer.Produce(ctx, h.topic, o.ID, payload); err != nil {

	}

//go:generate mockgen -source=./get_order.go -destination=./mock/get_order.go -package=queries_mock
type GetOrderInterface interface {
	Handle(ctx context.Context, id string) (*order.Order, error)
}

type GetOrderArgs struct {
	OrderRepository order.RepositoryInterface
	Cache           cacheclient.CacheInterface
}

type GetOrder struct {
	repo  order.RepositoryInterface
	cache cacheclient.CacheInterface
}

func NewGetOrder(args GetOrderArgs) *GetOrder {
	return &GetOrder{repo: args.OrderRepository, cache: args.Cache}
}

func (h *GetOrder) Handle(ctx context.Context, id string) (*order.Order, error) {
	key := consts.CacheKeyOrder(id)
	if raw, ok := h.cache.Get(key); ok {
		var cached order.Order
		if err := json.Unmarshal(raw, &cached); err == nil {
			return &cached, nil
		}
	}

	o, err := h.repo.FindByID(ctx, id)
	if err != nil {
		if errors.Is(err, errortype.ErrNotFound) {
			return nil, errs.ErrNotFound
		}
		return nil, errs.ErrDatabase
	}

	if raw, err := json.Marshal(o); err == nil {
		h.cache.Set(key, raw)
	}
	return o, nil
}

import (
	"context"
	"errors"
	"net/http"
	"os"
	"os/signal"
	"strconv"
	"syscall"
	"time"

	"github.com/sorrawichyooboon/integration-and-unit-testing-golang-v2/internal/adapters/kafkaclient"
	"github.com/sorrawichyooboon/integration-and-unit-testing-golang-v2/internal/config"

	"github.com/sorrawichyooboon/integration-and-unit-testing-golang-v2/internal/server"
	"github.com/sorrawichyooboon/integration-and-unit-testing-golang-v2/internal/utils"
)

func main() {
	log := utils.Logger()
	config.Init()

	ctx, stop := signal.NotifyContext(context.Background(), os.Interrupt, syscall.SIGTERM)
	defer stop()

	deps, err := server.InitCoreDependencies()
	if err != nil {
		log.Error("init dependencies", "err", err)
		os.Exit(1)
	}
	app := server.NewApplication(deps)

	// HTTP driving port.
	e := restapis.NewHttpServer(app)
	go func() {
		addr := ":" + strconv.Itoa(config.Current.Server.Port)
		if err := e.Start(addr); err != nil && !errors.Is(err, http.ErrServerClosed) {
			log.Error("http server", "err", err)
			stop()
		}
	}()

	// Kafka driving port.
	consumer := kafkaclient.NewConsumer(
		config.Current.Kafka.Brokers,
		config.Current.Kafka.ConsumerGroup,
		config.Current.Kafka.OrderCreatedTopic,
	)
	orderConsumer := consumers.NewOrderConsumer(app.Commands.ReserveStock)
	go func() {
		if err := consumer.Consume(ctx, orderConsumer.Handle); err != nil && !errors.Is(err, context.Canceled) {
			log.Error("consumer", "err", err)
		}
	}()

	<-ctx.Done()
	log.Info("shutting down")

	shutdownCtx, cancel := context.WithTimeout(context.Background(), 10*time.Second)
	defer cancel()
	_ = e.Shutdown(shutdownCtx)
	_ = consumer.Close()
	_ = deps.Producer.Close()
}

# integration-and-unit-testing-golang-v2

A complete **template** for structuring a Go service (hexagonal / ports-and-adapters)
and writing **unit tests** and **integration tests**. Example domain: Order & Inventory
(scaffold only — no real business logic).

## Layout

```
internal/
  ports/       driving adapters — restapis (Echo, oapi-codegen) + consumers (Kafka)
  services/    application core — queries (read) + commands (write)
  domains/     entities + repositories (+ sqlboiler models, errortype)
  adapters/    driven adapters — kafkaclient, cacheclient
  server/      composition root (dependencies + Application)
  config/ consts/ middleware/ utils/
errs/          AppError catalog + HTTP status mapping
pkg/           pure utilities (ptr, retry)
tests/ivt/     testcontainers integration suite
tests/kafka/   local Kafka for manual testing
```

## Testing

Two unit-test styles live side by side so you can compare them:

- **Ginkgo/Gomega BDD** (`*_test.go`) — primary; every unit-testable component.
- **Table-driven (stdlib `testing` + testify)** (`*_table_test.go`) — representative
  per layer: `commands.CreateOrder`, `queries.GetOrder`, `domains/order` repository,
  `pkg/retry`.

Mocking is `go.uber.org/mock` (gomock) throughout, generated via `//go:generate` into
`mock/` subfolders. Repository tests use `go-sqlmock`.

```bash
make tools-install     # install ginkgo, mockgen, oapi-codegen, sqlboiler, dbmate, golangci-lint
make unit-test         # Ginkgo + table-driven; excludes tests/ivt and generated models
make integration-test  # testcontainers: real Postgres + Kafka (requires Docker)
make lint
make generate          # oapi-codegen + sqlboiler + go generate (mocks) + go mod tidy
```

## Code generation

- HTTP server: `docs/server/openapi.yaml` → `internal/ports/restapis/api_gen/` (oapi-codegen).
- DB models: `db/migrations/*` applied to Postgres, then `sqlboiler.toml` →
  `internal/domains/db_models_gen/`.
- Mocks: `//go:generate mockgen ...` → `mock/` subfolders.

## Run locally

```bash
docker compose up -d          # Postgres + Kafka
export DATABASE_URL="postgres://postgres:postgres@localhost:5432/orders?sslmode=disable"
make migrate
go run .
```
